What is GitHub Pages?

GitHub Pages is a free static site hosting service that takes HTML, CSS, and JavaScript files directly from a repository on GitHub and publishes them as a website.

Features

  • Free hosting for static websites
  • Custom domain support
  • HTTPS enabled by default
  • Easy to set up and deploy
  • Perfect for portfolios, documentation, and project sites

Technologies Used

HTML5 CSS3 GitHub Pages

Getting Started

To create your own GitHub Pages site:

  • Create a repository on GitHub
  • Add your HTML files (starting with index.html)
  • Enable GitHub Pages in repository settings
  • Your site will be live at https://username.github.io/repository-name
